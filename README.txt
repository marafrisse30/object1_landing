Object 1 landing page template

Files:
- index.html
- styles.css
- script.js
- assets/ folder for images

Recommended image names:
- hero.jpg
- interior.jpg
- location.jpg
- gallery-1.jpg
- gallery-2.jpg
- gallery-3.jpg
- gallery-4.jpg
- gallery-5.jpg

How to use:
1. Put your real images into the assets folder with the names above.
2. Open index.html in a browser.
3. Change project name, payment plan, dates, phone and email in index.html.
4. Connect the form to your CRM/webhook later by editing the submit handler in script.js.
